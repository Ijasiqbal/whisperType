Vozcribe for Mac

Speak naturally. Get perfectly transcribed text — right where your cursor is.

Install via Homebrew

Recommended

$ brew tap Ijasiqbal/vozcribe $ brew install --cask vozcribe
Requires Homebrew. If you don't have it, paste this in Terminal first:
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

After installing

1

Open Vozcribe

Find it in your Applications folder or search with Spotlight (Cmd + Space).

2

Grant permissions

Allow Microphone access and Accessibility when prompted.

3

Start speaking

Use your hotkey to record. Transcribed text appears where your cursor is.

Requirements

macOS Ventura (13.0) or later
Apple Silicon or Intel Mac
Internet connection for transcription
